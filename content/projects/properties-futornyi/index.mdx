# Properties.futornyi.com - AI-Powered Real Estate Analysis Platform

A modern, intelligent platform that provides comprehensive AI-powered property analysis for the Greek real estate market. Built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui components.

## 🌍 What It Does

Properties.futornyi.com is a sophisticated landing page and demo platform that showcases AI-driven property analysis capabilities with a focus on the Greek real estate market and Golden Visa investment opportunities.
