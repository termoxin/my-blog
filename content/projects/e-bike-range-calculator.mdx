# E-Bike Range Calculator Project (April 2026)

This project now has **two connected apps**, each built for a different stage of ride planning and bike setup:

- **Main planner app:** [ebike.futornyi.com](https://ebike.futornyi.com/)
- **Physics playground:** [ebike.futornyi.com/playground](https://ebike.futornyi.com/playground)

The goal is simple: make battery decisions obvious before the ride starts, then let you test setup behavior in a live simulator.

## Two apps, one workflow

### 1) Main app — route-aware range planning
Use the main app when you want to answer practical questions like:
- Can I finish this route on one charge?
- How much does wind and climbing change my expected range?
- Where should I recharge on longer rides?

What it does best:
- GPX-based route analysis
- Weather-aware and elevation-aware energy estimates
- Battery consumption breakdown and charging-stop planning
- Browser-first workflow with private, local planning

Open it here: [ebike.futornyi.com](https://ebike.futornyi.com/)

### 2) Playground — real-time simulation and tuning
Use the playground when you want to experiment:
- How does changing motor/controller/battery setup affect ride behavior?
- What happens to dynamics under different assist, throttle, slope, and load?
- How does a specific scenario feel before trying it outside?

What it does best:
- Real-time simulation engine
- Component-level setup experimentation
- Scenario-driven testing and iteration
- Fast feedback loop for advanced users

Open it here: [ebike.futornyi.com/playground](https://ebike.futornyi.com/playground)

## How they work together
The main app helps you plan the actual ride outcome.  
The playground helps you understand and tune the machine behavior behind that outcome.

If you're planning a trip, start with the calculator.  
If you're comparing setups or exploring performance tradeoffs, move to the playground.
